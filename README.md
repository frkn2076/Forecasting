# Forecasting
Forecasting GBPUSD market price with LSTM model. Model updates itself with incremental learning every hour.

# Installments
* python 3.12.0
* MetaTrader 5

# Python dependencies
* pip install numpy
* pip install pandas
* pip install MetaTrader5
* pip install keras
* pip install scikit-learn
* pip install matplotlib
* pip install tensorflow
